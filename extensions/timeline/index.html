Description

Create beautiful interactive timelines using media from Twitter, Flickr, Google Maps, YouTube, Vimeo, Vine, Dailymotion, Wikipedia, SoundCloud and more.

Joomla Timeline is a Joomla implentation of Timeline JS by Knight Lab. See http://timeline.knightlab.com/#examples for examples.

Features

  • Create beautiful timelines in minutes
  • Load interactive media from various sources

Demo

Joomla Timeline uses Timeline JS. See http://timeline.knightlab.com/#examples for examples.
